CaféGest - Sistema de Gestión Cafetera

Sistema web completo para la gestión integral de parcelas cafeteras. Incluye módulos de recolección de café, facturación de ventas, control de insumos y dashboard principal con animaciones avanzadas. Cuenta con autenticación segura, generación de PDFs, exportación a Excel y diseño responsive.

Tecnologías: HTML5, CSS3, JavaScript ES6+, jsPDF, SheetJS, Netlify

Noticiero Digital MC

Noticiero Digital es una aplicación web que muestra noticias actualizadas de diversas categorías (general, negocios, entretenimiento, etc.) utilizando la API pública de News API. El sistema presenta una interfaz limpia y amigable para el usuario, con tarjetas de noticias, paginación y filtrado por categorías.

Tecnologías: PHP, Bootstrap 5, HTML5, CSS
